HTML - Basics

HTML (HyperText Markup Language) is the standard language used to create web pages. It defines the structure and layout using elements like headings, paragraphs, links, images, etc.

CSS - Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls fonts, colors, spacing, and responsive design across devices.

Core Java - Basics

Core Java includes basic concepts like Object-Oriented Programming, classes, objects, loops, arrays, and exception handling used in Java applications.

SQL - Basics

SQL (Structured Query Language) is used to manage data in relational databases. It includes commands like SELECT, INSERT, UPDATE, DELETE, and JOIN.

PHP - Basics

PHP (Hypertext Preprocessor) is a server-side scripting language used to build dynamic websites. It can interact with databases, handle forms, and manage sessions.
